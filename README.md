# Installing
1. Make a new folder (for example movieSlicer)
2. Download slider.py and place it in "movieSlicer" folder
3. make sure you have moviepy, numpy, pillow installed
4. ```
pip install moviepy==2.0.0.dev2 (or newer)
pip install Pillow
pip install numpy
```
5. Done
